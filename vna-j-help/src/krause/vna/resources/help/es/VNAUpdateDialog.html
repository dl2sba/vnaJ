General

Desde la versión 2.7 de vna / J se establece un procedimiento para que el usuario pueda comprobar si hay nuevas versiones y opcionales       descargar todos los archivos necesarios para la nueva versión en un directorio seleccionado

La función de actualización no envia ningún dato a Internet!

La función de actualización sólo carga el fichero mencionado en el campo Webseite y analiza su contenido.

Este archivo se descarga, cuando el botón Checch se pulsa.

Campos

Sirio de actualización

Esta es la URL del archivo de información de la versión en el servidor de actualizaciones.

Versión instalada

Está es la versión actual de vna/J.

Versión disponible

Está es la versión de vna/J en el servidor de actualizaciones.

Comentario

Un pequeño comentario de la nueva versión.

Contenido de ficheros

Una lista de todos los archivos necesarios para la versión está en el servidor de actualizaciones. El tamaño del fichero se muestra inicialmente como -1. El tamaño real del archivo se muestra, durante la descarga. El progreso de la descarga del archivo se muestra en la columna de estado.

Si la casilla de verificación Complete está seleccionada, todos los archivos, incluso aquellos actualmente no necesarios para el sistema operativo utilizado se descargan en el directorio de instalación. Si la casilla de verificación Complete no está seleccionada, sólo los archivos necesarios para el sistema operativo actual se descargan.

Esta casilla se trabaja a partir de V2.7.3 en adelante!

Directorio de instalación

Este es el directorio seleccionado de los archivos descargados. NO utilice el directorio actual, en el que vna/J se está ejecutando. Seleccione un directorio diferente. Todos los archivos necesarios para la nueva versión se pueden descargar en este directorio. La instalación actual no será modificada. Usted puede utilizar el PROPOSE botón para crear u nuevo nombre de directorio.

Botones

Check

Descarga el archivo que se muestra en el campo de actualización de sitio y muestra su contenido.

Proposito

Crear un nombre de directorio basado en el directorio de la aplicación que se está ejecutando y el nuevo número de versión

Asume que la aplicación actual está corriendo en c:/program files/vnaJ/2.6.15 y la versión de actualización disponible es 2.7.0q entonces el directorio propuesto será c:/program files/vnaJ/2.7.0q.

Completo

Al seleccionar esta casilla, descarga todos los archivos, independientemente del sistema operativo se está ejecutando actualmente.

Si no selecciona esta casilla, descarga sólo los archivos que se requieren para el sistema operativo actual.

Búsqueda

Se abre un diálogo de selección de directorio. Aquí se puede seleccionar la ubicación de todos los archivos descargados. También la creación de nuevos directorios está soportada.

Cerrar

Cierra este diálogo.

Ayuda

Esta ayuda.

Aborta

Este botón sólo se activa cuando las descargas están funcionando. Al hacer clic en este botón, se cancelan la descarga más actuales y todas las pendientes.

Instalación

Este botón se activa, si la versión vna/J en el sitio de actualización es más reciente que la versión vna / J ejecuta actualmente.

Al pulsar este botón se descarga todos los archivos que se muestran en el Lista de ficheros contenidos directorio mostrado en el campo Directorio de instalación
