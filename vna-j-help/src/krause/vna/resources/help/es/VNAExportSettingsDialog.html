General

Aqui pueden establecerse los parámetros generales para la exportación de datos.

Fichero de salida

Campo de entrada de nombre del fichero

Aquí puede introducir una plantilla para el nombre de archivo de los datos exportados. Cualquier número de variables puede ser insertado usando llaves, es decir {0} inserta la variable 0 en la cadena durante la exportación. Se admiten todas las variables de sustitución enumeradas mas adelante.

Casilla de verificación de sobreescritura

Si los archivos de exportación ya existen y la casilla de verificación está seleccionada, el archivo existente se sobrescribe automáticamente.Si la casilla no está seleccionada, se muestra un mensaje de advertencia.

Botón de búsqueda

Al hacer clic en los botones de búsqueda se abre un diálogo de selección de directorio, en el que el usuario puede seleccionar el directorio para los archivos exportados.

Título

Este título se imprime sobre el área de diagrama en la exportación de los formatos JPG y PDF. Están soportados los parámetros de sustitución igual que para los campos de nombre de archivo.

Comentario

Este comentario está impreso debajo del área de diagrama en el documento exportado a PDF. Están soportados los parámetros de sustitución igual que para los campos de nombre de archivo. El comentario se imprime con una fuente de tamaño fijo, por lo que puede ser formateado de forma sencilla.

Separador decimal

Por lo general, el separador decimal se selecciona en base a los parámetros del sistema operativo. Para la exportación del formato de parámetros-S, el separador decimal utilizado puede ser seleccionado aqui.

Variables soportadas

Estas variables son admitidas:

{0} La fecha/hora actual se inserta aquí. Además el formato de fecha/hora se describe en el siguiente capítulo.
{1} El modo de exploracióon seleccionado en el analizador.
{2} El nombre corto del analizador seleccionado.
{3} El nombre largo del analizador seleccionado.
{4} Frecuencia de arranque de exploración. Se muestran los separadores de miles.
{5} Frecuencia de parada de exploración. No se muestran los separadores.
{6} Frecuencia de arranque de exploración. Se muestran los separadores de miles.
{7} Frecuencia de parada de exploración. No se muestran los separadores.
{8} Número de muestras usadas para esta exploración.
{9} El número de muestras usadas para generar los datos (todavía no soportado).
{10} El número de muestras del conjunto de datos de calibración.
{11} El número de exploraciones usadas para generar el conjunto de datos de calibración.
{12} Nombre del conjunto de datos de calibración.
{13} El nombre del usuario actual en la operación.

Cuando se llama a la función de exportación, la hora actual es proporcionada en el parámetro {0}. A continuación se muestran algunos ejemplos de formateo de marcado de tiempo 2010-02-15 17:12:45:

VNA_{0,date,yyMMdd} --> VNA_100215.xls

VNA_{0,time,HHmmss} --> VNA_171245.xls

VNA_{0,date,yyMMdd}_{0,time,HHmmss} --> VNA_100215_171245.xls
