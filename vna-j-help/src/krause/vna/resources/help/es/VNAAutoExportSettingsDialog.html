Ayuda en ajustes de AUTO-Exportación

General

Fichero de salida

Campo de entrada de nombre de fichero

Aqui puede introducir una plantilla para el nombre del fichero de los datos exportados.

Estas variables son soportadas:

{0}
Aqui es insertada la fecha/hora actuales.
{1}
El modo seleccionado de exploración del analizador.
{2}
El nombre del analizador seleccionado.
{3}
El nombre largo del analizador seleccionado.
{4}
Frecuencia de comienzo de exploración. Se muestran los separadores de miles.
{5}
Frecuencia de parada de exploración. No se muestran los separadores.
{6}
Frecuencia de comienzo de exploración. Se muestran los separadores de miles.
{7}
Frecuencia de parada de exploración. No se muestran los separadores.
{8}
Número de muestras usadas en esta exploración.
{9}
El número de exploraciones usadas para generar los datos(Todavía no soportado).
{10}
El número de muestras en el conjunto de datos de calibración.
{11}
El número de exploraciones usadas para generar el conjunto de datos de calibración.
{12}
Nombre del conjunto de datos de calibración.
{13}
El nombre del usuario actual de la operación.

Casilla de verificación de sobreescritura

Si los archivos de exportación ya existen y la casilla de verificación está seleccionada, el archivo existente se sobrescribe automáticamente. Si la casilla de verificación no está seleccionada, se muestra un mensaje de advertencia.

Botón de búsqueda

Al hacer clic en los botones de búsqueda se abre un diálogo de selección directorio, en el que el usuario puede seleccionar el directorio para los archivos exportados.

Cabecera

Esta cabecera se imprime sobre el área de diagrama en formato JPG y en la exportación a PDF. Los parámetros de sustitución iguales que para los campos de nombre de archivo están soportados.

Comentario

Este comentario está impreso debajo del área de diagrama en el documento PDF exportado. Los parámetros de sustitución iguales que para los campos de nombre de archivo están soportados. El comentario se imprime con una fuente de tamaño fijo, por lo que se puede formatear de forma fácil.

Separador decimal

Por lo general, el separador decimal se selecciona en base a los parámetros del sistema. Para la exportación de formato parámetros-S, el separador decimal utilizado puede ser seleccionado aquí.
