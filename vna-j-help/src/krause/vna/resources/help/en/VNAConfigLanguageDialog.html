Help on language selection

Use this dialog to configure the language settings of vna/J.

Usually vna/J tries to determine the locale of the computer it runs. Sometimes it may be necessary to switch to another language. If vna/J is used on a computer, whose language is not supported by vna/J, the default language English is selected.

Based on the language settings, i.e. formatting of numbers is done as defined by this language.

After changing the language settings inside vna/J an application restart is necessary.

vna/J currently is translated into the languages for which a radio button exists in the dialog.

Buttons

Systen
vna/J tries to determine the correct language from the systems language settings
Other
Always use this for the language settings.
Help
Displays this help dialog.
Close
Closes this dialog without saving the changes.
OK
Saves the changes and ask the user for an application restart.
