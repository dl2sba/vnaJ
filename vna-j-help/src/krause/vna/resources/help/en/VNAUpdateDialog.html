General

Since version 2.7 of vna/J a procedure is established, to enable the user to check for new versions and optional download all files required for the new version into a selected directory

The update function didn't send any data to the internet!

The update function only loads the file mentioned in the field Webseite and analyses its content.

This file is downloaded, when the Check button is pressed.

Fields

Update site

This is the URL of the version information file on the update server.

Installed version

This is the currently running version of vna/J.

Available version

This is the version of vna/J on the update server.

Comment

A short comment for the new version.

Contained files

A list of all files required for the version on the update server. Filesize is initially displayed as -1. The effective filesize is shown, during download. The progress of the file download is shown in the status column.

If the checkbox Complete is set, all files, even those currently not needed for the used operating system are downloaded in the selected installation directory. If the checkbox Complete is not set, only those files needed for the current operationg system are downloaded.

This checkbox works from V2.7.3 onwards!

Installation directory

This is the selected directory for the downloaded files. Do NOT use the current directory, in which vna/J is running. Select a different directory. All required files for the new version will be downloaded to this directory. The current installation will not be touched. You can use the PROPOSE -button to build a new directory name.

Buttons

Check

Download the file displayed in the update-site field and display its content.

Propose

Build a directory name based on the directory of the currently running application and the new version number

Assume current application is running in c:/program files/vnaJ/2.6.15 and the available update version is 2.7.0q then the proposed directory name would be c:/program files/vnaJ/2.7.0q.

Complete

Selecting this checkbox, downloads all files, regardless of the current operation system running.

Unchecking this box, downloads only those files, which are required for the current running operation system.

Search

Opens a directory selection dialog. Here the location for all downloaded files can be selected. Also creation of new directories is supported.

Close

Closes this dialog.

Help

This help.

Abort

This button is only enabled, when downloads are running. Clicking this button cancels the current and all further downloads.

Install

This button is enabled, if the vna/J version on the update site is newer then the currently running vna/J version.

Pressing this button downloads all files displayed in the List of contained files to the directory displayed in field Installation directory
