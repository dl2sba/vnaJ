Usage

This dialog can be used i.e. to tune an antenna filter to a given centre frequency and a defined band-width.

The marker-math dialog is bound to one of the markers. The data displayed in this dialog is the data of this marker.

Mode

The mode used depends on the data at the selected cursor location.

If the data left and right of the cursor is lower, than PEAK-mode is selected.

If the data left and right of the cursor is higher, than NOTCH-mode is selected.

Limit

A limit in dB must be defined, which is used to search for the low- and high-limits.

Return-/Transmission-Loss

If the marker should use the Return-Loss scale, the RL-radio button must be selected.

If the marker should use the Transmission-Loss scale, the TL-radio button must be selected.

Sample - tune a bandpass-filter

Assume you want to tune a BP to a 6dB-bandwidth of 100kHz ...

  1. Set one of the markers to the desired center-frequency of the bandpass-filter

  2. Enter 6 in the Limit field.

  3. Now tune the filter until the desired bandwidth of 100kHz is displayed in the Bandwidth-field .
